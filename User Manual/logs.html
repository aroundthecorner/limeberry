Creating Logs & Log Class




Logger

This class is used to generate logs. You can use the functions of class for creating Error files, warning files or output messages. Created txt files will be located in 'output' folder of your application.


Output()

Create an output log file. 

Logger::Output(LogTypeProvider  $type, string  $title,string  $msg ) : void

$type:
LogTypeProvider::Error() OR LogTypeProvider::Warning() OR LogTypeProvider::OutputMessage()

$title: title for your output log
$msg: contents for log

Namespace

If you want to use this library, you need to import required namespace in your php code file. Required namespace:


use limeberry\io\Logger;

use limeberry\io\LogTypeProvider;
