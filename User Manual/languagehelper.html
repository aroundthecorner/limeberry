Language Helper Class

Limeberry framework supports multi language application support. Using this helper class you can create multi language applications.


Example Language File

Add new language file in your {application}/language folder. In this example we will create a language file called default.en.php contents of file; 

< ?php

/**
 * EXAMPLE LANGUAGE FILE FOR OUR APPLICATION
 * DEFAULT TERMS FOR LANGUAGE: ENGLISH.
 */


$panel = array("lblDash" => "Dashboard",
               "lblLogin" => "Login", 
               "lblCreate" => "Create",
               "lblDelete" => "Delete",
               "lblUpdate"=>"Update");

$messages = array("msgSuccess"=> "Your new user record is saved to database",
                  "msgError"=>"Error while saving new user to database",
                  "msgWarnin"=>"Warnin; please check the following fields to continue");


isAvailable()

This method is a static method used to check f a language file available. Returns bool 

Language::isAvailable(string  $prm_file) : bool

$prm_file File name of your language file located in language path


setLanguage()

This method is used to set a language file to work with. File must be in language folder 

$lang = new Language();
$lang->setLanguage(string  $prm_file) : $this

$prm_file File name of your language file located in language path


setVariable()

Check and set variable in language file to get the contents. 

$lang = new Language();
$lang->setLanguage("default.en.php")->setVariable(string  $variable_name) : $this

$variable_name Variable name in default.en.php to get.


bindLanguage()

Get contents of file set in previous methods. 

$lang = new Language();
$lang->setLanguage("default.en.php")->setVariable("panel")->bindLanguage() : Array




Example

Lets create two language files English and Turkish, and load the greeting message based on the parameter in action.
default.en.php 

< ?php

/**
    * Language: English
    */

$data = array("msg1"=>"Welcome to my website");

default.tr.php 
< ?php

/**
* Language: Turkish
*/

$data = array("msg1"=>"Web siteme hosgeldiniz");


Now in our indexController.php file and in indexAction we will create a logic, if parameter is "tr" we will load the default.tr.php or if the parameter is "en" we will load the default.en.php file. 

public function indexAction($param="en")
{      
    //create class instance
    $lang = new Language();
    
    //create file name based on parameters, may be sessions/cookies or configuration from
    //database for user based settings.
    $langFileName = "default.".$param.".php";
    
    //Create an array with returned data
    $contents = $lang->setLanguage($langFileName)->setVariable("data")->bindLanguage();
    
    //use language
    echo $contents["msg1"];
    
}

Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\helpers\Language;
