Models

Model is a class which designed to help you with information in your database. We can call it 'Hand' of human body, because when you request a data they should bring information from database. Also when you say to a model to hold this information untill I put it to database, model classes should hold that information.
All your model class files must be located in 'model' folder, limeberry look this directory for autoloading a model file. You can read more about folder structure.
DO NOT FORGET; It is not required but recommended to save your models with Model suffix. Because when you call a Model called User.php from a controller called User.php they both will have a class User.




Load()

Load is a static method to locate and load a model. 

Model::Load(string  $prm_file);

$prm_file File name of your model file
