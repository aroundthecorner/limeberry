Pin Checker Class

A pin chechker class for taking control and managing operations. This class is a derivative of Object Mapper class

Map new pins 

PinChecker::MapNew(string  $pinname, mixed  $pincode) : void

$pinname: Name for your value
$pincode: Pin to be stored

CheckPin()

Check if pin matches with given data and continue if true. 

PinChecker::GetValue(string  $pinname, mixed  $pincode) : boolean

$pinname: Name for your pin
$pincode: Stored pin data
