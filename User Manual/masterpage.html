Template Engine

limeberry master pages allows you to create layouts for your web pages in your application.It is designed to eliminate the need to write the same html code for each repetitive design every time.Once you design a template, you can easily relate it to all the pages and accelerate your coding experience to twice that. This topic can be thought of as the continuation of the 'view' topic you can read more about views Here.


Define New Page

This is an example view file using template engine. You should save your views in project_folder/application/view directory. 

< ?php
use limeberry\Page;

$view = new Page();
$view->SetLayout("master.php");


$view->BeginContent();
?>

< !-- YOUR HTML OR PHP CODES -->

EndContent(); ?>


How Template Engine of limeberry Works

Template Engine Language

If you want to use template engine, you need to follow some rules. Some of them are like the ones below;

  1. Content Place Holder: Please do not forget to define and echo $this->contents variable where you want to put your view's html contents. For example in your template file you need to type; echo $this->contents;

  2. Define Parameters: If you want to set parameters between layouts and views, you need to type {@parameter1} in your layout file. You can set a value for your parameters by typing $view_variable->SetParameter("parameter1", "my value is..");

  3. Page Title : You can set a different title for every view. You can complete this task by setting a parameter but also if you type and echo $this->title; on your template file you can use and set a title by SetTitle command



SetLayout()

This function is used to set template file for your view. 

$view->SetLayout(string  $prm_file);

$prm_file File name of your template file



SetParameter()

Set content for parameter defined in template page. 

$view::SetParameter(string  $prm_name,string  $prm_value );

$prm_name Parameter name available in template
$prm_value value you want to set



SetParameterNull()

Set null value to unused parameter in template files 

$view->SetParameterNull(string  $prm_name);

$prm_name Parameter name available in template



SetTitle()

if you do not use parameters for setting title you can use this function. Remember < ?php echo $this->title; ?> must be defined in template. 

$view->SetTitle(string  $title);

$title Title for your page



includeFile()

Include a css, js or img file in your template or views. This function returns a relative path for template folder. 

Page::includeFile(string  $prm_file);

$prm_file Name of your file
