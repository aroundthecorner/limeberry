FTP Class

You can use limeberry framework's ftp class to connect to a ftp server.


__construct

Initialize ftp class 


use limeberry\tool\ftp as limeberry_ftp;
$myftp = new limeberry_ftp($host,$user,$password, $port= 21, $timeout);
@param string Ftp Host
@param string Ftp account user
@param string Ftp account password
@param int Ftp port, default 21
@param int Ftp connection time out, default 100

Connect

Connect to ftp. @return bool 

use limeberry\tool\ftp as limeberry_ftp;
$myftp = new limeberry_ftp($host,$user,$password, $port= 21, $timeout);
$myftp->Connect();

Close

Close the ftp connection. @return void 

use limeberry\tool\ftp as limeberry_ftp;
$myftp = new limeberry_ftp($host,$user,$password, $port= 21, $timeout);
$myftp->Connect();
//your ftp operations...
//...
$myftp->Close();

Error

Returns last error of ftp action as String. 

use limeberry\tool\ftp as limeberry_ftp;
$myftp = new limeberry_ftp($host,$user,$password, $port= 21, $timeout);
$myftp->Connect();
//your ftp operations...
//...
echo $myftp->Error(); 
$myftp->Close();

Download

Download a file from ftp server.
@param string Local file path for download
@param string Remote path and file
@return bool

use limeberry\tool\ftp as limeberry_ftp;
$myftp = new limeberry_ftp($host,$user,$password, $port= 21, $timeout);
$myftp->Connect();
//your ftp operations...
//...
$myftp->Download("C:\downloads\readme.txt", public_html\w\read_.txt);
$myftp->Close();

Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\tool\Ftp;
