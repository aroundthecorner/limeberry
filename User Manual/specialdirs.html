Limeberry Special Directory Helper Class

You can access all paths of app in code.

ApplicationFolder()

Returns application folder path. 

SpecialDirectory::ApplicationFolder() : String

FrameworkFolder()

Returns limeberry folder path. 

SpecialDirectory::FrameworkFolder() : String

ControllerFolder()

Returns controller folder path. 

SpecialDirectory::ControllerFolder() : String

ConfigFolder()

Returns config folder path. 

SpecialDirectory::ConfigFolder() : String

LibraryFolder()

Returns library folder path. 

SpecialDirectory::LibraryFolder() : String

LanguageFolder()

Returns language folder path. 

SpecialDirectory::LanguageFolder() : String

ModelFolder()

Returns model folder path. 

SpecialDirectory::ModelFolder() : String

OutputFolder()

Returns output folder path. 

SpecialDirectory::OutputFolder() : String

TemplateFolder()

Returns template folder path. 

SpecialDirectory::TemplateFolder() : String

ViewFolder()

Returns view folder path. 

SpecialDirectory::ViewFolder() : String

Example

Let's read all contents of view/index.php file and print it to screen $data = limeberry\io\FileSystem::ReadFile(limeberry\io\SpecialDirectory::ViewFolder().DS."index.php"); var_dump($data);

Namespace

Warning To use this helper class you need to use

use limeberry\io\SpecialDirectory;
