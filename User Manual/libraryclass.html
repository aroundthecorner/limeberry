Library Class

Library class is used to include third party class and code files to your project. It is simply a replacement for php's include and require functions.


Load()

Loads a third party class. (Class must be in library folder) 

Library::Load(string  $filename, IncludeType  $type) : void

$filename: filename in 'library' directory
$type: ModeRequire, ModeRequireOnce , ModeInclude, ModeIncludeOnce , ModeDefault

LoadConfig()

Loads a limeberry configuration file 

Library::LoadConfig(string  $filename, IncludeType  $type) : void

$filename: filename ex; 'path\file.php'
$type: ModeRequire, ModeRequireOnce , ModeInclude, ModeIncludeOnce , ModeDefault

isAvailable()

Checks if file is available 

Library::isAvailable(string  $filenam) : boolean

$filename: filename ex; 'path\file.php'

Example

                        
if(Library::isAvailable('library\bcrypt.php'))
{
    Library::Load("bcrypt.php");
}

Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\tool\Library;
