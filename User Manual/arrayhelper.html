Array Helper Class

This is a helper class for arrays. It's adds various features to arrays


getElement()

Select one or more element from an array. 

Arrays::getElement(Array   $array_var, Mixed   $vars ) : Mixed

$array_var Selected array to operate with.
$vars Array | String (array key) ;searched value(s) in $array_var.


setEmpty()

Sets element(s) of array to empty. Sets the selected keys in an array to empty string 

Arrays::setEmpty(Array   $array_var, Mixed   $vars ) : Void

$array_var Selected array to operate with.
$vars Array | String (array key) ;searched value(s) in $array_var.


setNull()

Sets element(s) of array to Null. Sets the selected keys in an array to NULL 

Arrays::setNull(Array   $array_var, Mixed   $vars ) : Void

$array_var Selected array to operate with.
$vars Array | String (array key) ;searched value(s) in $array_var.


Reverse()

Reverse an array horizontally or vertically 

Arrays::Reverse(Array   $array_var, String   $option ) : Array

$array_var Selected array to operate with.
$option REVERSE_HORIZONTAL | REVERSE_VERTICAL

Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\helpers\Arrays;
