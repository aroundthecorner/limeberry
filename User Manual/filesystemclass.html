FileSystem Class

This class is used to run commands related File System.



isAvailable()

Checks if a file exists in a path 

FileSystem::isAvailable(String $path) : boolean

$path: File Name

ReadFile()

Read all contents of a file 

FileSystem::ReadFile(String $path) : String

$path: File Name

WriteFile()

Write to file 

FileSystem::WriteFile(String $path, String $contents) : boolean

$path: File Name
$contents: Contents to write in $path

Append()

Append to file 

FileSystem::Append(String $path, String $contents) : boolean

$path: File Name
$contents: Contents to write in $path

Delete()

Delete a file 

FileSystem::Delete(String $path) : boolean

$path: File Name

Copy()

Copy a file 

FileSystem::Copy(String $path, String $path_to) : boolean

$path: copy file from
$path_to: copy file to

getItemList()

Returns an array of items in path 

FileSystem::getItemList(String $path, string  $opt = SCANDIR_SORT_NONE ) : Array

$path: Directory to list

getSize()

Size of file 

FileSystem::getSize(String $path) : integer

$path: file name

getPermissions()

Returns permissions for a file 

FileSystem::getPermissions(String $path) : integer

$path: file name

getOwner()

Returns owner of file 

FileSystem::getOwner(String $path) : Mixed

$path: file name

getType()

Get type of file 

FileSystem::getType(String $path) : Mixed

$path: file name

getTotalSpace()

Get total space available in dir 

FileSystem::getTotalSpace(String $path) : Mixed

$path: directory


Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\io\FileSystem;
