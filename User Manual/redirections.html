Routes & Redirections

limeberry framework uses controllers to map your routes. You does not required to specify routes when you codding your page relationships.You can read URL Structure of limeberry before reading and trying redirections.

Let's say you have 2 controllers User and Post
in UserController.php controller we have 3 Actions 

NewAction
UpdateAction(userID)
DeleteAction(userID)
and in PostController.php we have 2 Actions 
ShowAction(postID)
UpdateAction(postID)
In this case we have the following routes 
http://localhost/User/New
http://localhost/User/Update/ID
http://localhost/User/Delete/ID


http://localhost/Post/Show/ID
http://localhost/Post/Update/ID



RedirectToAction()

This function is used to redirect to actions in controllers. RedirectToAction function gets 3 parameters and returns string. 

 Url::RedirectToAction(string  $controllerName, string  $action,  mixed $parameter = null) : string



Example 
use limeberry\Url as pUrl;
< a href="< ?php  echo pUrl::RedirectToAction("User","Update", "9282"); ?>  "> User Name < /a>

< a href="< ?php  echo pUrl::RedirectToAction("User","New"); ?>  "> Create User < /a>

< a href="< ?php  echo pUrl::RedirectToAction("Post","Show", "7892"); ?>  "> Read Topic < /a>

...

AreaRedirection()

This function is used to redirect to actions in sub folders in controller path. AreaRedirection function gets 4 parameters and returns string. 

 Url::AreaRedirection(string  $area, string  $controllerName, string  $action,  mixed $parameter = null) : string

Redirect()

Simple redirect url; For example you can use this with html's a tags href attribute 

 url::Redirect(string  $url)

$url; Link you want to redirect; ex: Redirect("http://www.example.com")

Example; Use In Controllers

In this example when we navigate to http://localhost/todo_list/Index/Show the ShowAction redirects us to IndexAction in Index controller with parameter 999. 

use limeberry\Controller As p_ctrl;
use limeberry\Url as redirect;

class indexController extends p_ctrl{

    public function indexAction($param=0)
    {		

        if($param == 1)
        {
            $this->View->our_header = "This is list: ".$param;
            $this->View->our_text = "text text text text text text text ";
        }else if ($param==2)
        {
            $this->View->our_header = " Second Item  ";
            $this->View->our_text = "second second second second second second second second second ";
        }else{
            $this->View->our_header = "Item ".$param;
            $this->View->our_text = $param;
        }

        ##Load view
        if($this->View->isAvailable("home/home.phtml"))
        {
            $this->View->Render("home/home.phtml");
        }else{
            echo "Page not found";
        }
    }
    public function ShowAction($param=0)
    {		
        header('location: '.redirect::redirectToAction("Index", "Index", "999"));
        echo "	Redirecting.....";
    }
}

GetData()

You can use GetData method to get a form data posted with get method. 

 Url::GetData(string  $id)

$id; (optional)* If you do not set this value, you will get an array with all values, You can set it to get a specific value from posted Url data.

 Url::GetUrl()



Route Class

This class is used to simply manage routes of your application. All functions of this class must be used in Application Register Function

Route Class:: Required Namespace

If you want to use this library, you need to import required namespace in your php code file. Required namespace: use limeberry\Route;

Route Class:: Available Methods

Route::MapNew()

You can use this function to map a new url. 

 Route::MapNew("login-page", "/controller/action/params1../paramsN");



Route::ResolveMap()

This function returns a value of mapped url data. 

 Route::ResolveMap("login-page");
returns: "/controller/action/params1../paramsN"



Route::ClearMaps()

This function clears all mapped url data. 

 Route::ClearMaps();



Route::ForceRedirect()

This function is used to redirect your application with headers. 

 route::ForceRedirect("name-of-mapped-url", "value-to-set-if-parameter-is-available");


Example use with parameters: 

//Set url in ApplicationClass
Route::MapNew("searchpage", "/applications/search/{@}");
.
