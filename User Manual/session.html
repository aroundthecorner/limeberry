Session Class

limeberry has a light session class. You can use limeberry framework's session function to manage your session data.


__construct

Initialize ftp class
@param string Optional name for session class
@return void

use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");

Create

Creates new session data

@param string id for session
@param string value for session
@return void

use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");
$mysession->Create("user_type","admin");

Delete

Deletes a session if available

@param string Session id
@return bool

use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");
$mysession->Create("user_type","admin");
$mysession->Delete("user_type");

Clear

Clear all session data set. 

use limeberry\tool\session;
..
..
$mysession = new session();
$mysession->Create("user_type","admin");
$mysession->Clear();

Destroy

Destroy all sessions with php's session_destroy command. 

use limeberry\tool\session;
..
..
$mysession = new session();
$mysession->Create("user_type","admin");
$mysession->Destroy();

isAvailable

Checks if session is available

@param string session id
@return bool

use limeberry\tool\session;
..
..
$mysession = new session();
$mysession->Create("user_type","admin");

if($mysession->isAvailable("user_type"))
{
echo "session is set.";
}else{
echo "Not Set";
}

GetName & SetName

Using this two functions you can set a session class name get session class name. GetName @return string 

use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");
$mysession->Create("user_type","admin");
echo $mysession->getName();


SetName Change name of current session class
@param string session class name
@return void
use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");
$mysession->Create("user_type","admin");
$mysession->setName("other-name-for-class");

CreateMultiple

Create sessions with multiple values
@param array key for session id and value for session data
@return void

use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");
$mysession->CreateMultiple(array("user_type"=>"admin", "login_time"=>"14:30"));

DeleteMultiple

Delete multiple session data with arrays and keys.
@param array Session id's as array
@return bool

use limeberry\tool\session;
..
..
$mysession = new session("testing-session-class");
$mysession->DeleteMultiple(array("user_type", "login_time"));

Regenerate

Generate new id for current session class
@param bool true if old session data will be stored
@return void

$mysession = new session("testing-session-class");
$mysession->Regenerate(false);




Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\tool\Session;
