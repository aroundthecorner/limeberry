String Helper Class

This is a helper class for strings. It's adds various features to srings


Define new String

Store a new string data. 

$name = new String("username");



get()

Get stored data 

$name = new String("username");
$name->get();



set()

Set new data to variable 

$name = new String("username");
$name->set("new updated data");



toUpper()

Convert stored data to uppercase 

$name = new String("username");
$name->toUpper();
//returns: USERNAME



toLower()

Convert stored data to lowercase 

$name = new String("USER");
$name->toLower();
//returns: user



toUpperFirst()

Convert stored strings first char to uppercase 

$name = new String("username");
$name->toUpperFirst();
//returns: Username



toLowerFirst()

Convert stored strings first char to lowercase 

$name = new String("USER");
$name->toLowerFirst();
//returns: uSER



addPrefix()

Adds prefix for each word in string 

$name = new String("php, mvc, framework");
$name->addPrefix("updated_", ","); //second parameter is limiter to explode the string
//returns: updated_php updated_mvc updated_framework



addSuffix()

Adds prefix for each word in string 

$name = new String("php   mvc   framework");
$name->addSuffix("_used", " "); //second parameter is limiter to explode the string
//returns: php_used mvc_used framework_used



addSfxPrx()

Adds prefix and suffix for each word in string in the same time 

$name = new String("php-mvc-framework");
$name->addSfxPrx("}","{", "-"); //addSfxPrx(suffix, prefix, limiter);
//returns: {php} {mvc} {framework}


Namespace


If you want to use this library, you need to import required namespace in your php code file. Required namespace:

use limeberry\helpers\String;
